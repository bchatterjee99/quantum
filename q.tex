\begin{verbatim}

Half-Adder:
-----------

            +---------------+                    +-------------------+
            |               |                    | a | b | a^b | a&b |               
     a ---->|               |----> a ^ b         +---|---|-----------|               
            |               |                    | 0 | 0 |  0  |  0  |               
            |       HA      |                    | 0 | 1 |  1  |  0  |               
            |               |                    | 1 | 0 |  1  |  0  |               
     b ---->|               |----> a & b         | 1 | 1 |  0  |  1  |               
            |               |                    +-------------------+               
            +---------------+
